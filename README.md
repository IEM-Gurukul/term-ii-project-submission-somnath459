# PCCCS495 – Term II Project

## Project Title

Student Attendance Management System using Java

## Problem Statement (max 150 words)

Managing student attendance manually using registers is time-consuming and prone to errors. Teachers often face difficulty in recording attendance for large numbers of students daily and tracking previous records. Manual systems also make it hard to organize, maintain, and analyze attendance data efficiently.
This project proposes a Student Attendance Management System developed using Java and Object-Oriented Programming (OOP) concepts. The system provides a digital solution where teachers can add student details, mark daily attendance, and easily access records. It reduces paperwork, saves time, and improves accuracy. Additionally, it helps in generating summaries of attendance, making the process more efficient and reliable.

## Target User

Teachers and school/college administrators who manage student attendance records.

## Core Features

-Add and manage student details
-Mark daily attendance (Present/Absent)
-View student attendance records
-Search student by ID or name

## OOP Concepts Used

- Abstraction:Abstract classes or interfaces are used to define common behaviors like attendance operations.
- Inheritance:The Student class inherits properties from a base class such as Person.
- Polymorphism:Methods behave differently based on objects, such as displaying student details or attendance reports.
- Exception Handling:Handles invalid inputs, missing data, or runtime errors to ensure smooth execution.
- Collections / Threads:Java Collections like ArrayList and HashMap are used to store and manage student and attendance data efficiently.

---

## Proposed Architecture Description

The system follows an object-oriented design approach. It consists of multiple classes such as Student, Attendance, and Main.
-The Student class stores student details like ID and name.
-The Attendance class manages attendance records.
-Data is stored using Java collections like ArrayList or HashMap.
-The Main class provides a menu-driven interface where users can perform operations like adding students, marking attendance, searching records, and generating summaries.

## How to Run

-Compile the Java files.
-Run the Main class.
-Use the menu options to interact with the system.
